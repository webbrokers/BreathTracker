# BreathTracker

BreathTracker — лёгкий MVP‑трекер дыхания, который использует микрофон ноутбука для отслеживания вдохов и выдохов, визуализирует дыхательную волну и рассчитывает показатели (BRPM, длительность фаз, I:E, когерентность). В репозитории лежит статический HTML‑файл `breath-mvp-light.html`, который можно открыть прямо в браузере или задеплоить на любой хостинг статических сайтов.

## Быстрый старт

1. Откройте файл `breath-mvp-light.html` в современном браузере (Chrome/Edge/Firefox с поддержкой Web Audio API).
2. Разрешите доступ к микрофону после нажатия кнопки **Start**.
3. Отрегулируйте слайдеры: время калибровки, коэффициент EMA, чувствительность детектора и усиление входного сигнала.
4. Следите за графиком амплитуды, маркерами вдох/выдох и таблицей метрик; используйте кнопку **Export CSV** для выгрузки сессии.
5. Для дыхательных практик можно включить метроном «6 вдохов в минуту» (кнопка в правой панели).

## Работа в GitHub Codespaces

В репозитории есть конфигурация `.devcontainer/devcontainer.json`, которая разворачивает контейнер на базе `mcr.microsoft.com/devcontainers/javascript-node:1-20-bullseye` с предустановленными расширениями VS Code (ESLint, Prettier, Tailwind CSS и т. д.). После старта Codespace:

- Откройте `breath-mvp-light.html` через Live Server или встроенный превьюер.
- Соблюдайте настройки форматирования (Prettier, LF, табы по 2 пробела).

## Текущие ограничения

- MVP работает локально без сборщика и тестов; любые изменения нужно проверять вручную в браузере.
- Используется Web Audio API и AudioWorklet; для мобильных браузеров требуется HTTPS.
- На результаты влияют шум и системные фильтры; для точности используйте тихое помещение или гарнитуру.
